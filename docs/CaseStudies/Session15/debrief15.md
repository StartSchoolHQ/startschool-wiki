## Defrief: Key Takeaways

### Never Do This:

* Leave default database permissions in production
* Store sensitive documents without encryption
* Ignore security warnings from your tools
* Skip metadata removal from user uploads
* Launch without security reviews

---

### Always Do This:

* Change default security settings before going live
* Remove or encrypt sensitive data
* Test your security with external tools
* Have security experts review your code
* Plan for what happens if data is breached
